# MPL Source Offer

This distribution includes files under MPL-2.0.

For every modified MPL-covered file in this repository, complete corresponding source code is provided in this repository history and release artifacts.

If you received binaries without source, request source code by opening an issue in this repository and include:

- Product/version identifier
- Binary package checksum
- Target platform information

Source code will be provided for at least 12 months from the binary release date.
